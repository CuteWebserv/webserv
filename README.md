## 프로젝트 개요

HTTP 2.0 기반의 Webserver 입니다.
기본적인 GET, POST, PUT, DELETE 요청 및 CGI 요청을 수신 및 반환 합니다.

## 실행 방법
1. **설치**: 프로젝트를 클론
```bash
  git clone https://github.com/CuteWebserv/webserv.git
```
2. **실행**: 프로젝트를 실행
```bash
  cd /webserv
  make
  # 주의사항: gcc98, makefile 라이브러리가 필요합니다!!
```
